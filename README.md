# 家庭目标管理系统

一个基于Next.js和React的家庭任务与奖励管理系统，帮助家长管理孩子的日常任务，并通过星星奖励系统激励孩子完成任务。

## 功能特点

- 🏠 多家庭支持，数据隔离
- 👨‍👩‍👧‍👦 区分家长和孩子角色
- 📋 家长可以添加、删除任务，并分配给特定孩子
- ✅ 孩子可以将任务标记为完成
- ⭐ 任务完成后获得星星奖励，带有精美的星星动画效果
- 🎊 完成所有任务后触发庆祝动画
- 📊 星星累计统计与获取日志
- 👥 家长可以查看所有家庭成员的星星统计
- 🔄 自动保存登录状态，支持会话保持

## 技术栈

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## 安装与运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 打开浏览器访问 [http://localhost:3000](http://localhost:3000)

## 使用说明

### 登录
在登录页面选择一个用户角色（家长或孩子）进行登录。系统支持家庭邮箱和PIN码验证，并自动保存登录状态。

### 家长功能
- 添加新任务：设置任务标题、描述、奖励星星数量，并分配给特定孩子
- 删除任务：可以删除任何任务，删除前会有确认提示
- 查看星星统计：查看所有家庭成员的星星总数和获取日志
- 管理孩子任务：可以切换查看不同孩子的任务列表

### 孩子功能
- 完成任务：点击"完成"按钮将任务标记为已完成，获得星星奖励
- 查看星星统计：查看自己的星星累计和获取日志
- 任务完成反馈：完成任务时会有星星动画效果，完成所有任务时会有庆祝动画

## 数据隔离

系统通过familyId实现家庭数据隔离，不同家庭的用户只能看到自己家庭的任务和星星记录。

## 角色权限

- **家长**：可以添加、删除任务，查看所有家庭成员的星星统计
- **孩子**：可以完成任务获取星星，只能查看自己的星星统计
